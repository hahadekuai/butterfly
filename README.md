Butterfly
=========

## 简介

butterfly专注于前端最佳解决方案，致力于成为前端的spring

它不仅仅是一些js模块，而是希望从多方面最佳地去解决前端开发上的问题，让我们可以快乐高效地开发产品。

目前版本: 1.1.0

它包括四个组成部分

1. Framework
2. DPL
2. 工具和环境
3. 文档库
