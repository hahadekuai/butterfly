Marquee ======= ## 描述 滚动效果，鼠标hover停止滚动 ## API ~~~js new Marquee(element, config) ~~~ ## config
名称 类型 默认值 说明
body string .fx-body 滚动元素的选择器
effect string left 滚动效果，可选值有 left, right, up, leftright
stopOnHover boolean true 鼠标hover时是否停止滚动
step number 4 速度, 每次滚动偏移(px), 值越大，滚动越快
speed number 80 每次滚动时间间隔(ms)。值越大，滚动越慢
delay number 1000 每轮滚动结束后停止时间(ms)
