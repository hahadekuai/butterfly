Popup ======= ## 描述 浮出效果, 可以用来实现菜单，浮层等效果 ## API new Popup(element, config) ## config
名称 类型 默认值 说明
trigger string .fx-popup-trigger 触发节点
showTrigger string 显示触发节点选择器，默认和trigger相同
hideTrigger string 隐藏触发节点选择器，默认和trigger相同
showEvent string mouseenter 显示事件
hideEvent string mouseleave 关闭事件
body string .fx-popup-body 内容节点选择器
effect 
	
1. string
2. { show: function(elm, callback), 
hide: function(elm, callback)}
default 浮出效果
active string fx-popup-active 显示时具有的className
showDelay number 0 延迟多少ms打开
hideDelay number 0 延迟多少ms关闭
