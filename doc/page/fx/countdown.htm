CountDown ======= ## 描述 用于实现倒计时计数 ## API new CountDown(element, config) ## config
名称 类型 默认值 说明
beforeShow string .fx-countdown-unstart 倒计时没有开始时会展示在effectClass元素上的选择器
inProcessShow string .fx-countdown-inprocess 倒计时进行中会展示在effectClass元素上的选择器<
finishedShow string .fx-accordion-finish 倒计时结束后会展示在effectClass元素上的选择器<
startTime string 无 开始时间,例如:'2012-12-12 09:01:00'
endTime string 无 结束时间,例如:'2012-12-12 09:11:00'
