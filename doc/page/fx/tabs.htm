Tabs ======= ## 描述 用于实现切换效果 ## API ~~~js new Tabs(element, config) ~~~ ## config
名称 类型 默认值 说明
nav string .fx-tab-nav li nav选择器
pane string .fx-tab-pane 面板选择器
active string active 当前选中项className
prev string .prev 上一页选择器
next string .next 下一页选择器
disabled string disabled 上一页/下一页不可点时className
index number 0 当前选中项索引
event string mouseenter 切换触发事件
interval number 3000 自动切换时间间隔
effect 
1. string

2. function(o, callback)

3. {
	setup: function(o), 
	show: function(o, callback) 
}
default 切换效果, 内置有 default, leftright, topdown, fade
duration number 动画时间
autoSwitch string | function(now, length) 自动切换规则
before [event] function(o) 切换前触发，返回false阻止切换
switch [event] function(o) 切换后触发
