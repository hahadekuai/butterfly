Accordion ======= ## 描述 用于实现手风琴切换效果 ## API new Accordion(element, config) ## config
名称 类型 默认值 说明
nav string .fx-accordion-header nav选择器
pane string .fx-accordion-content 面板选择器
navHoverClass string fx-accordion-hhover 鼠标滑过nav部分时会添加的class名
navActiveClass string fx-accordion-hactive nav被激活时会添加的class名
next string .next 下一页选择器
effect string updown 手风琴的类型,支持'updown','leftright'
duration number 500 切换所花费的时间
index number 0 当前选中项索引
event string mouseenter 切换触发事件
