Paging ====== ## 描述 用于动态分页条的创建 ## API ~~~js new Paging(div, options) ~~~ ## config
名称 类型 默认值 说明
render boolean false 是否由js渲染分页条
listSize number 7 分页条页码数目，当render为true时有效
showForm boolean false 是示显示分页表单，当render为true时有效
showPrevNext boolean true 是示上一页下一页，当render为true时有效
pageIndex number 初始选中的页面，如果不指定则从页面结构推导
pageCount number 总页数, -1表示未知分页
goTo function(page) 分页动作, 返回false, 忽略本次跳转
## DEMO ### 分页条由后端渲染 #### HTML 可参考DPL [paging](http://wpstatic.china.alibaba.com/doc/index.htm?_#doc/dpl/paging) ~~~html <div class="ui-paging" data-url="paging_.htm?page={page}"> <div class="paging"> <ul class="paging-list fd-clr"> <li><a href="paging_.htm?page=1" class="pre pre-disabled">上一页</a></li> <li><a href="paging_.htm?page=1" class="current">1</a></li> <li><a href="paging_.htm?page=9">9</a></li> <li><a href="paging_.htm?page=4" class="omit">...</a></li> <li><a href="paging_.htm?page=2" class="next">下一页</a></li> </ul> <div class="paging-form fd-clr"> <span>共<em class="pagenum">3</em>页</span> <span>到<input type="text" class="ui-txt ui-txt-small pnum" autocomplete="off" maxlength="4">页</span> <a title="确定" href="#" class="paging-btn">确 定</a> </div> </div> </div> ~~~ #### js ~~~js define(['jQuery', 'ui.Paging'], function($, Paging) { var context = document, elm = $('div.ui-paging', context); new Paging(elm); }); ~~~ #### 效果 ### 分页条由前端渲染 #### html ~~~html <div class="ui-paging paging-01"></div> <br /><br /> <div class="ui-paging paging-02"></div> <br /><br /> <div class="ui-paging paging-03"></div> ~~~ #### js ~~~js define(['jQuery', 'ui.Paging'], function($, Paging) { var elm = $('div.paging-01', document); new Paging(elm, { render: true, renderForm: true, pageCount: 47, goTo: function(page) { // do paging } }); var elm = $('div.paging-02', document); new Paging(elm, { render: true, pageCount: -1, goTo: function(page) { // do paging } }); var elm = $('div.paging-03', document); new Paging(elm, { render: true, pageCount: 10, showPrevNext: false, goTo: function(page) { // do paging } }); }); ~~~ #### 效果
