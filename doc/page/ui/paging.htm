Paging ====== ## 描述 用于动态分页条的创建 ## API ~~~js new Paging(div, options) ~~~ ## DEMO ### 分页条由后端渲染 #### HTML 可参考DPL [paging](http://wpstatic.china.alibaba.com/doc/index.htm?_#doc/dpl/paging) ~~~html <div class="ui-paging" data-url="paging_.htm?page={page}"> <div class="paging"> <ul class="paging-list fd-clr"> <li><a href="paging_.htm?page=1" class="pre pre-disabled">上一页</a></li> <li><a href="paging_.htm?page=1" class="current">1</a></li> <li><a href="paging_.htm?page=9">9</a></li> <li><a href="paging_.htm?page=4" class="omit">...</a></li> <li><a href="paging_.htm?page=2" class="next">下一页</a></li> </ul> <div class="paging-form fd-clr"> <span>共<em class="pagenum">3</em>页</span> <span>到<input type="text" class="ui-txt ui-txt-small pnum" autocomplete="off" maxlength="4">页</span> <a title="确定" href="#" class="paging-btn">确 定</a> </div> </div> </div> ~~~ #### js ~~~js define(['jQuery', 'ui.Paging'], function($, Paging) { var context = document, elm = $('div.ui-paging', context); new Paging(elm); }); ~~~ #### 效果 ### 分页条由前端渲染
