ColorChooser ============ ## 描述 PlaceHolder ## API new PlaceHolder(element, config) ## config
名称 类型 默认值 说明
placeholder string placeholder字符
## DEMO
## 用法 ### 使用Autowire ~~~html <input type="text" data-placeholder="请输入您的用户名" data-widget-type="ui.PlaceHolder" /> ~~~ ### 使用Javscript ~~~js define(['jQuery', 'ui.PlaceHolder'], function($, PlaceHolder) { var input = $('input.mytext'); new PlaceHolder(input, { placeholder: '请输入您的用户名' }); }); ~~~
