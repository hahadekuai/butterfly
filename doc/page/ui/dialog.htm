Dialog ============ ## 描述 浮出层组件 ## API new Dialog(element, config); Dialog.warn(msg, title, options); Dialog.error(msg, title, options); Dialog.info(msg, title, options); Dialog.confirm((msg, title, options)); ## config
名称 类型 默认值 说明
center boolean true 是否居中
closable boolean true 有无关闭按扭
draggable boolean,selector true 是否可拖动, 以及拖动handler
content string 对话框内容
buttons true 按钮 
[{
	'name': 'confirm',
	'text': '确定'
}]
showHeader boolean true 是否显示头部, 默认情况下如果有title就会显示header
title string 显示标题
showFooter boolean 是否显示底部，默认情况下根据是否有buttons来决定是否显示footer
className string 添加到dialog节点的class, 可以使用它定义dialog的样式
width number 对话框宽度
height number 异步加载时,对话框初始高度, 用于改善交互
forceHeight boolean 默认情况下, height仅仅是对话框打开时的高度，后来内容载入后就会自定义的,如果设置forceHeight, 则会设置d-content的高度
loader function 内容载入器
beforeOpen function 打开窗口前触发, 返回false阻上对话框打开
beforeClose function 关闭窗口时前触发，可以做一些清理工作
contentSuccess function 内容载入完成时触发（已更新到对话框节点中，可以操作DOM了）
## DEMO 代码 ~~~js $('a.dialogtest').on('click',function(e){ e.preventDefault(); new Dialog({ title:'tip', content:'just a test', buttons:[{ 'name': 'confirm', 'text': 'confirm' },{ 'name': 'cancel', 'text': 'cancel' }] }); }); $('a.dialoginfo').on('click',function(e){ e.preventDefault(); Dialog.info('a info demo', 'title'); }); $('a.dialogwarn').on('click',function(e){ e.preventDefault(); Dialog.warn('a warn demo', 'title'); }); $('a.dialogerror').on('click',function(e){ e.preventDefault(); Dialog.error('a error demo', 'title'); }); $('a.dialogconfirm').on('click',function(e){ e.preventDefault(); Dialog.confirm('a confirm demo', 'title',{ confirm:function(){ alert('confirm sending'); } }); }); ~~~ demo common demo info demo warn demo rror demo confirm
