ColorChooser ============ ## 描述 颜色选择组件 ## API new ColorChooser(element, config) ## config
名称 类型 默认值 说明
value string 当前值
transparent boolean false 是否允许选择透明
## DEMO
## 用法 ~~~js define(['jQuery', 'ui.ColorChooser'], function($, ColorChooser) { var box = $('div.box', 'div.color-chooser-demo'); box.on('click', function() { new ColorChooser(box, { value: '#ff0000', transparent: true, confirm: function(o) { box.css('background', o.color); } }); }); }); ~~~
