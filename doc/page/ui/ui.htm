UI ===== ## 描述 提供图片缩放，ie6 position fixed等支持 ### UI.resizeImage ~~~js /** * @param {selector|element} selector 需要缩放的图片 * @param {number | options} options|size 最长边大小 */ UI.resizeImage(selector, options) ~~~
名称 类型 默认值 说明
size number 缩放后的最大边, 等比缩放
width number 缩放后的宽
height number 缩放后的高
force 默认情况下当图片小于指定大小时不进行缩放，指定force强制缩放
defaultImage string 默认图
### DEMO

~~~js define(['jQuery', 'ui.UI'], function($, UI) { UI.resizeImage($('div.image-box-01 img', 'div.ui-demo'), 100); }); ~~~ ### UI.positionFixed 对ie6 position fixed进行特殊处理 ~~~js UI.positionFixed(div, options) ~~~
名称 类型 默认值 说明
bottom boolean false 是否使用bottom进行定位
show function(elm) jQuery#fadeIn 显示逻辑
